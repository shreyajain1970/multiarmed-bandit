# Multi-Armed Bandit RL Project

## Description
Implemented Random, Epsilon-Greedy, Softmax, and UCB agents for the Multi-Armed Bandit problem. Tracks average reward and % optimal action.  

## Files
- `bandit_environment.py` – Bandit environment class  
- `agents.py` – All four agents  
- `experiment.py` – Runs experiments and generates plots  

## Plots
- Average Reward vs Steps  
- % Optimal Action vs Steps
